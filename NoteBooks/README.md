# Notebooks Directory

This directory contains Jupyter notebooks for data exploration and analysis.

## Notebooks

- `cancer_detection_pipeline.ipynb` - Main analysis notebook including:
  - Data cleaning and preprocessing
  - Unsupervised learning (Isolation Forest, LOF, Autoencoder)
  - Supervised learning (SVM, XGBoost)
  - Model evaluation and visualization

## Running Notebooks

```bash
jupyter notebook
```

Or use VS Code with the Jupyter extension.

## Best Practices

- Keep notebooks clean and documented
- Use markdown cells for explanations
- Clear outputs before committing
- Consider using nbstripout to remove outputs automatically

# Scripts Directory

This directory contains Python scripts for data processing and utilities.

## Scripts

- `add_binary_label.py` - Adds binary labels (Cancer/Healthy) to the dataset

## Usage

Run scripts from the project root directory:

```bash
python scripts/add_binary_label.py
```

## Adding New Scripts

When adding new scripts:
- Use clear, descriptive names
- Add docstrings and comments
- Handle errors gracefully
- Use relative paths for file access
